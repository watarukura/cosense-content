# お作法
ユニケージ開発手法におけるお作法

> シェルスクリプトは記述の自由度が高い。そのため、ユニケージではシェルスクリプトの標準的処方や原則を決め、開発手法としての統一を図っている。これを「お作法」と呼んでいる。
ユニケージ原論 p.67
