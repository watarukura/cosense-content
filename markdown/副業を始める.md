確定申告

    - MoneyForwardクラウドで帳簿を作る
        - 経費はMoneyForward Meから確定申告フラグを立てて取り込む
            - この際、新聞図書費にしたいので、カテゴリを趣味・娯楽ではなく教養・教育にしておくとよい
            - 電気代とかなんとかはめんどくさいので諦めた
            - 自宅用にMacBook Airを買ったので経費に入れた(消耗品費)
                - 30万円未満なので一括償却できる
        - Amazon.co.jpから領収書をPDF化する
            - MoneyForwardクラウドへのアップロード時に5MB以下という制限があるため、ghostscriptを使ってサイズを縮小する
                - brew install ghostscript しておくこと
            - 領収書の一括ダウンロード機能とかつけてほしいが、どの注文番号にどの商品が含まれるか、みたいなのもあるし簡単ではなさそう
                - この辺もMoneyForward Meで面倒見てくれればいいのになー
                    - [Q. 「証憑自動取得」機能を使って「Amazon.co.jp」から「支払明細書」を取得することはできますか？](https://biz.moneyforward.com/support/account/faq/features/20220524.html)
                        - できるんじゃん！？
        - 売上は請求書から作る
            - 入金を確認して消込する
    - MoneyForwardクラウドアプリから申告する
        - 入力はWebからでもできるのでこっちのほうが楽だが、申告自体はアプリから行う
            - なので、サブスクリプションはAppStoreで行うのが良く、パーソナルミニにした
        - [https://gyazo.com/318571909835a0f6d9461ec582f60e78](https://gyazo.com/318571909835a0f6d9461ec582f60e78)
        - iDeCo、生命保険料控除の証明書から転記する
            - e-Taxまたはアプリからの申請の場合、証明書自体の送付は不要だが、5年保持する義務あり
            - [https://www.e-tax.nta.go.jp/toiawase/qa/kakutei/tempu01.htm](https://www.e-tax.nta.go.jp/toiawase/qa/kakutei/tempu01.htm)
        - 新NISAも1800万円超えた分は収入扱いになるのかな？
            - まだしばらくそんな金額にならないので、様子見

```gs.bash
#!/usr/bin/env bash
set -euo pipefail

input=$1
/usr/local/Cellar/ghostscript/10.02.1/bin/gs \
    -sDEVICE=pdfwrite \
    -dPDFSETTINGS=/ebook \
    -dNOPAUSE \
    -dQUIET \
    -dBATCH \
    -sOutputFile="$input-圧縮.pdf" "$input"

```
