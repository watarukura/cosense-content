# Level1データ
[https://uec.usp-lab.com/JOURNAL/CGI/JOURNAL.CGI?POMPA=DOC_journal09](https://uec.usp-lab.com/JOURNAL/CGI/JOURNAL.CGI?POMPA=DOC_journal09)
> LEVEL1は生明細データであり、テキスト形式とは限らない。LEVEL1は非同期で発生することも多い。これを毎秒、毎分、毎時、毎日のように同期的にシステムに取り込み、情報処理のタイムテーブルに乗せたものがLEVEL2となる。 通常、LEVEL2はテキスト化され、イレギュラーデータは取り除かれる。

例

    - アップロードされたExcelファイルやCSVファイル
- 受信したEDI注文データ
- ECサイトから入力された注文フォーム
- POSレジから登録された売上データ
- ・・・etc
