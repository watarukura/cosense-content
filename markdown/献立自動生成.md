# 献立自動生成
[watarukura.icon](watarukura.icon)どうやったらできるかな

献立に求めるもの

- 材料が揃っていること または 材料の不足がわかること
    - 今、冷蔵庫に入っていて、そろそろ賞味期限が切れそうなものを使いたい
- 自分の技量で作れること
    - マグロの柵を切る、はできるが、鯵を三枚におろす、は自分ではできない
- 十分な栄養が取れること
    - 赤・黄・緑の栄養素が、一定期間内でバランスよく配置されている
- 特定の食材が除外されること
    - 子供の年齢により摂取してはいけない食品がある
    - 妻が摂取したがらない類の食材がある
- 旬の食材が使われていること
    - 安い
    - 栄養価が高い

献立自動生成の要素

    - レシピ
        - 調理難易度表示
    - 食材の在庫管理
    - 位置情報 x 食材の旬の時期情報
    - 食材の栄養価情報
    - (赤ちゃん向けの場合) 食材禁忌情報
    - (大人向けの場合) 食材の好き嫌い情報登録
    - 年代別の栄養価必要量情報
