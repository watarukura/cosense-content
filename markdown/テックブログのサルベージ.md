以下のツールを利用した

- [Online Cosense Converter](https://hashrock.github.io/md2sb-online/)
    - markdown -> cosenseへの変換に使用
- [Save to Cosense (ex-Scrapbox) - Chrome ウェブストア](https://chromewebstore.google.com/detail/save-to-cosense-ex-scrapb/jcdhmhfihdilhhnjhflmanmoimfjpakh)
    - web scrapに使用
    - cosenseにページを作成し、元ページへのリンクと指定したハッシュタグが設定される
