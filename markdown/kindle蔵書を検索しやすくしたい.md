# kindle蔵書を検索しやすくしたい

- 書籍の検索がめんどい
  - scrapboxは検索を減らせる
    - リンクが勝手につながる
- ダウンロードめんどい
  - マンガ本棚はダウンロードしなくていい
    - 残念ながらコミック以外は未対応
    - 技術書は会社の経費でリアル本を買うことにする
    - 小説はファイルサイズが小さいのでダウンロードする
- マンガ本棚はASINがURLに含まれている
  - マンガ本棚の一覧画面からkindleコンテンツのタイトルを取得する
  - マンガ本棚の一覧を全部開く
    - devtoolsからチョチョイとして1000個くらいタブを開いた
  - [OneTab](https://chrome.google.com/webstore/detail/onetab/chphlpgkkbolifaimnlloiipkdnihall?hl=ja)で開いたタブをURL一覧に変換する
  - URLからASINを取得する
    - GSheetにURLを貼って関数で取得した
- scrapboxで蔵書管理してみる
  - 書影がほしい
    - ASINからISBNを引いてOpenBDから書影を取得しようと思っていた
      - ASINで検索するためProduct Advertisement APIを実行しようとしようとアソシエイト・プログラムに申し込む
      - ぐぐったら書影＋アソシエイト・プログラム対応画像取得できそうだったのでありがたく借用
        - Amazon書影＋Affiリンク
      - →残念ながら、アソシエイト・プログラムの審査NGだった(2019/11/05)
        - 書影の取得方法を変更する必要がある
  - シリーズでまとめたい
    - やり方を考える
