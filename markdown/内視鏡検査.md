# 内視鏡検査
胃と大腸の内視鏡検査をした

    - ピロリ菌除菌したので、毎年胃の内視鏡検査をしている
    - 寝てる間に終わる、と聞いて[新宿内視鏡クリニック](https://www.shinjuku-naishikyo.com/)を予約した
    - ついでなので一回やっておくといい、と妻のススメで大腸内視鏡検査も同時に実施

タイムライン

- 2022/05/28(土) 予約
    - 希望日を5つくらい出す必要があり、絞り出すのがむずい
    - 結局、別途メールで調整した
- 2022/06/11(土) 事前診察
    - 検査を受けたい理由を問診票に書く
        - 保険診療となるため、それらしいことを書く必要がある
        - 便秘がある、とかで良いとのこと
    - iPadで説明動画を視聴
    - 血液検査
    - 腹部エコー
    - 下剤を2日分受け取る
- 2022/06/20(月) 夕食後、下剤1
- 2022/06/21(火) 夕食後、下剤2
    - 野菜・果物・辛いもの・コーヒー等はNG
    - プレーンな蒸しパン、豆腐、素うどん、具なし味噌汁、卵雑炊など、消化によいものを3食食べる
    - 水分は水で
    - 21時には寝る
- 2022/06/22(水) 検査
    - 朝食NG
    - 胃内視鏡検査
        - 静脈から鎮静剤を投与
            - 名前を呼ばれたら答える、を数回繰り返すようだ
            - 1回分しか記憶がない
        - 意識がない間に口からカメラを飲んで検査される
            - 目が覚めたあとも特に痛みがなく、腕の良い方にあたったようだ
        - 検査前に下剤3を飲む
    - 便がほぼ水になるまで10回以上トイレに行く
        - 1時間程度かかった
        - 色が所定の閾値を超えたらナースコールしてみてもらう
    - 大腸内視鏡検査
        - 再度鎮静剤を投与
            - 1回分も記憶がない
    - 検査結果説明
        - 写真を見ながら説明を受ける
    - 会計
        - 事前に説明があるが、検査時にポリープがあれば切除する
            - この場合は手術となるため費用アップ
            - 今回はポリープはなかったので検査費用のみ
        - 2週間程度でメールで結果の通知があるとのこと

感想

    - 検査前日は三食雑炊とヨーグルトとウィダーインゼリーで過ごした
        - ものすごい腹が減る
    - 検査前日は下剤の影響で睡眠が細切れになり、トイレとの往復になる
    - 検査当日、朦朧となる時間は10分程度
        - すぐに回復してすごい
    - トイレに行き過ぎて＊が痛くなって辛かった
        - 特に痩せないのに筋肉は落ちて理不尽
    - 検査後は腸内フローラの復活のために新ビオフェルミンS生活をしている
