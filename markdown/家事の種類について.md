# 家事の種類について
一括りにすると不幸が発生するので分けてみる

- 手を動かす家事 
- [食器洗い](食器洗い)
- [日常の洗濯](日常の洗濯)

- 覚えておく家事
- 消耗品の買い物
- 季節の掃除
- 衣替え

- 考える家事
- 料理の献立作り

- 自動化が容易なのは手を動かす家事で、覚えておく家事、考える家事の自動化は難しい
- ロボット掃除機、食洗機、洗濯乾燥機などでは「覚えておく」「考える」家事の負担は軽減できない
- 単独ではなく複合する場合もある
- 草むしりをするのは手を動かす家事だが、どの程度茂ってきたら草むしりをするかの閾値を決めるのは考える家事だし、定期的に実施するなら覚えておく家事になる
- 自分のことだけ考えて家事をする場合でも手間がかかるが、家族がいる場合は更に家族の健康や文化の程度に影響する
- レシピサイトで献立作りを支援することはできるが、冷蔵庫に何があって、賞味期限が切れそうな食材は何で、料理をする人がどの程度の調理の技量があって、直近で似たようなものを食べてなくて、といった複数のパラメータを満たすのは容易ではない
- 子供がいる場合は月齢ごとに食べてはいけない食材があり、アレルギーなども考慮する必要がある
