      - [‎HIIT & Tabata Timer -simple App - App Store](https://apps.apple.com/us/app/hiit-tabata-timer-simple/id1166542235) を使っている

      - 6km/hで大体45−60min歩きながら雀魂してると30minくらいで汗だく
    - ようやく懸垂が一回はできるようになった
      - まだデッドハングの状態からはできないんだけど
      - 広背筋をもっと鍛えたい

      - mi bandと一緒に使ってたアプリがバージョンアップしたときに体組成計に非対応になって、iPhone 17にしたら使えなくなった...
      - 体組成計、なんかいいのないかなー

仕事

  - 1-3月
    - 転職活動がうまくいかない中、急性虫垂炎で入院
      - 副業先だったDMMの上長に泣きついて「一緒に働かせてください」と懇願、無事入社
      - もし、次に転職ってなったときはとりあえず業務委託から始めたい
        - 質問への回答が思わしくなかったときに、目から興味が失われる感じ、マジで凹むので
          - 2025-01-28 - 2025-02-03 ここで書いてた
      - 現職への拾ってもらった恩をちゃんと返せるようにがんばる
    - 2月 無職になる
      - 無職期間が1ヶ月あるために国民年金・国民健康保険に切り替えが必要で超面倒だった...
      - トラーナと元トラーナの同僚の会社で副業を開始
        - シェルスクリプト + terraform + GASでプロトタイプを作成
          - シェルスクリプト→Pythonに書き換え
    - 3月から新入社員
      - 週1勤務から週5勤務に切り替わっただけで特に仕事の内容は変わってない
  - 4-6月
    - フルリモート → 月4出社へ
      - 出社はポジティブ、通勤はネガティブ
    - AIツールのハンズオンをいっぱいやる
      - なぜオレ...???
      - まぁ、最近Cursor + MCPハンズオンをやったPdMのチームから「使ってますよ！」って話が出て嬉しかった
    - 副業
      - GCloudでBigQuery + CloudRunみたいな構成にして動かす
      - テストを生成してリファクタリングコードも生成する
        - JetBrainsのAIアシスタントが活躍した
  - 7-9月
    - terraformの新規リポジトリを3つくらい作った
    - 新卒さんのフォローに入ったり
  - 10-12月
    - MBPを交換してもらった 
      - 久々に新Macセットアップ202512したので、ついでにdotfilesもきれいにした
        - [https://github.com/watarukura/dotfiles](https://github.com/watarukura/dotfiles)
        - Nixデビューもした
        - fish + vim → zsh + neovimに移行してみた
    - 新規事業のインフラ構築をぱぱっとやる、みたいな仕事が多かった
      - クラウドインフラ屋さんが足りてない感がある...
    - 社の負荷試験ツールのシナリオを書いて実際に動かして負荷を掛けてみた
      - 自前でインフラも面倒みてた身からするとすげー楽だった
  - 全体
    - 得意なことを自分で手を動かして成果を上げられる状況で大変ありがたい
      - 一方で、年齢とともに市場価値が下がっていくことを受け入れないといけない
      - どこかで年収が下がるので心の準備をする
    - AIな一年だった
      - 2026年は更に加速する気もするが、気をつけてないとバーンアウトするので適度に美味しいところだけつまんでいきたい
